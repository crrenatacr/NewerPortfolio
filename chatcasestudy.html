React Chat App

Overview

Built a simple yet fully functional chat app using React Native. The goal? Allow users to send text messages, record audio, and take pictures with a camera—all in one smooth experience.

About the project

Chat App is a mobile messaging application built with React Native and Expo, designed for real-time communication. The app allows users to chat with friends and family, share images, and exchange location data. It also features offline message reading, making it a reliable communication tool. The app ensures accessibility for users with visual impairments by supporting screen readers.

Purpose & Context

Chat apps are everywhere, but I wanted to build my own—because why not? The goal was to create a smooth, modern messaging app using React Native with text, audio, and photo-sharing capabilities. Oh, and I wanted to keep things simple while learning more about navigation, permissions, and Expo’s magic tricks. Also, it was a very important topic of my Full Stack Web Development Course.

Objective

Develop a fully functional chat app that allows users to:

✔ Send and receive text messages
✔ Record and send voice messages
✔ Capture and share photos
✔ Seamlessly navigate between screens

All while keeping the app lightweight, smooth, and (hopefully) crash-free.

Duration and credits

This project took longer than expected due to merging and debugging challenges, which were addressed with guidance from my course mentors. I deeply appreciate their support and teamwork throughout the process.
