Globalgigs Meet App

Case Study

About the project

This project is a serverless, progressive web application (PWA) with React using a test-driven and behavior-driven development (TDD/BDD) technique. The application uses the Google Calendar API to fetch upcoming events and presents some charts with event info.

Overview

Globalgigs is a Progressive Web App (PWA), with FrontEnd developed in React, with the ability to work offline and a serverless backend developed using TDD and BDD techniques for unit, integration and end-to-end testing. Users of this app are able to use it whenever they want to view upcoming events for a specific city. The application uses the Google Calendar API to fetch the events of an specific calender and AWS Lambda functions to execute authentication and authorization using the users Google account.

Purpose & Context

Globalgigs React Meet App was a personal project built as part of my Full Stack Web Development course at CareerFoundry to enhance my on-learning skills following best practices in programming.

Objective

The goal of this project was to develop a robust full-stack application that enhances my portfolio while deepening my understanding of integrating and optimizing various features seamlessly.

Duration and credits

This project took longer than planned, as it wasn’t possible to finish it on time for merging and debugging questions, which were revised during and after with my course mentors, whom I thank for all the time spent helping me with tips and decisive team work.
